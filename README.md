# Ngrok 内网穿透一键配置脚本

使用Ngrok的隧道，一键转发内网的Http/TCP端口

> PS: 由于Ngrok免费版本的限制，本脚本只能转发HTTP和TCP协议

## 使用方法

```shell
wget -N https://raw.githubusercontent.com/lauren12133/Ngrok-1key/master/ngrok.sh && sh ngrok.sh
```

快捷方式 `bash ngrok.sh`
